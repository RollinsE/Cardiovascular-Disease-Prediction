This project explores and models a public cardiovascular disease dataset to predict the presence of heart disease using a range of classical machine learning methods. The workflow covers data ingestion from Google Drive, exploratory data analysis (EDA), data cleaning, feature engineering, model development, evaluation, and interpretability.

Dataset: https://drive.google.com/file/d/1cd5wvAvJjdKtdFhaIpyMl75h2ory1w6O/view?usp=sharing
